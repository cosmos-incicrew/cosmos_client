# BSTI 설계 문서

피부 MBTI. **20문항 → 4축 채점 → 16개 유형 → 유형별 성분·자차 추천**

| 항목 | 내용 |
|---|---|
| 동작 방식 | 프론트(Flutter)에서 전부 처리 — **백엔드·네트워크 호출 없음** |
| 코드 위치 | `lib/features/bsti/` |
| 검증 | 테스트 **28개 통과** |
| 데이터 규모 | 유형 16 · 문항 20 · 보기 80 · 성분 34 · 성분매핑 216 · 자차 6 · 근거논문 15 |

---

## 1. 전체 흐름

```
답변 20문항  →  축별 점수 합산  →  cutoff 비교  →  4글자 코드  →  유형 데이터 조회
   {1:4, 2:3…}      oil / sens        ≥ cutoff?      "OSPW"       페르소나 · 성분 · 자차
                    pigment / aging   높은극/낮은극
```

---

## 2. 4개 축

문항 점수 합이 **cutoff 이상 → 높은 극**, 미만 → 낮은 극

| 축 | 코드 | 높은 극 | 낮은 극 | 문항수 | cutoff |
|:--|:--|:--|:--|--:|--:|
| 유·수분 | `oil` | **O** 지성 | **D** 건성 | 5 | 12.5 |
| 민감도 | `sensitivity` | **S** 민감 | **R** 저항 | 6 | 15.0 |
| 색소 | `pigment` | **P** 색소 | **N** 비색소 | 4 | 10.0 |
| 노화 | `aging` | **W** 주름 | **T** 탱탱 | 5 | 12.5 |

- **cutoff = 2.5 × 문항수** (중간값)
- **코드 조합 순서**: oil → sensitivity → pigment → aging

**예시**

| 축 | 점수 합 | cutoff | 판정 |
|:--|--:|--:|:--|
| oil | 13 | 12.5 | **O** |
| sensitivity | 16 | 15.0 | **S** |
| pigment | 11 | 10.0 | **P** |
| aging | 14 | 12.5 | **W** |
| | | | **⇒ OSPW** |

## 3. 문항 · 채점

| 항목 | 값 |
|---|---|
| 총 문항 | 20개 |
| 문항당 보기 | 4개 |
| 보기 점수 | 1~4점 (**4 = 높은 극 O/S/P/W 방향**) |
| 축별 배정 | oil 5 · sensitivity 6 · pigment 4 · aging 5 |

**문항 예시**

| # | 축 | 문항 | 4점 | 3점 | 2점 | 1점 |
|:--|:--|:--|:--|:--|:--|:--|

| Q6 | sensitivity | 최근 4주간 홍조·따가움이? | 자주 있었음 | 가끔 | 드물게 | 전혀 없음 |
| Q12 | pigment | 잡티·기미·주근깨가 있나요? | 많아요 | 보통 | 조금 | 거의 없음 |

## 4. 16개 유형

### 지성 (O) 계열

| 코드 | 페르소나 | 케어 순서 | 자차 |
|:--|:--|:--|:--|
| **OSPW** | 진정 먼저 챙기는 풀코스 케어 고양이 | 진정·장벽 → 선크림 → 미백 → 안티에이징 | 틴티드(젤) |
| **OSPT** | 피지 잡고 순하게 톤업하는 고양이 | 진정·장벽 → 선크림 → 미백 | 틴티드(젤) |
| **OSNW** | 장벽부터 쌓고 주름 대비하는 고양이 | 진정·장벽 → 선크림 → 안티에이징 | 무기자차(젤) |
| **OSNT** | 수분만 챙기는 속예민 물광 고양이 | 진정·장벽 → 선크림 | 무기자차(젤) |
| **ORPW** | 미백·주름 풀가동하는 강철 고양이 | 피지·모공 → 선크림 → 미백 → 안티에이징 | 틴티드(젤) |
| **ORPT** | 피지·잡티 잡는 탄탄 톤업 고양이 | 피지·모공 → 선크림 → 미백 | 틴티드(젤) |
| **ORNW** | 안티에이징에 올인하는 광채 고양이 | 피지·모공 → 선크림 → 안티에이징 | 유기자차 |
| **ORNT** | 기본만으로 완성되는 꿀피부 고양이 | 피지·모공 → 선크림 | 유기자차 |

### 건성 (D) 계열

| 코드 | 페르소나 | 케어 순서 | 자차 |
|:--|:--|:--|:--|
| **DSPW** | 장벽부터 차근차근 쌓는 종합케어 고양이 | 진정·장벽 → 선크림 → 미백 → 안티에이징 | 틴티드(크림) |
| **DSPT** | 보습 위에 미백 얹는 예민 진정 고양이 | 진정·장벽 → 선크림 → 미백 | 틴티드(크림) |
| **DSNW** | 수분으로 주름 늦추는 레이어링 고양이 | 진정·장벽 → 선크림 → 안티에이징 | 무기자차 |
| **DSNT** | 보습 하나로 빛나는 미니멀 고양이 | 진정·장벽 → 선크림 | 무기자차 |
| **DRPW** | 보습 깔고 잡티·주름 공략하는 전략 고양이 | 보습·장벽 → 선크림 → 미백 → 안티에이징 | 틴티드(크림) |
| **DRPT** | 선크림이 무기인 차분한 톤업 고양이 | 보습·장벽 → 선크림 → 미백 | 틴티드(크림) |
| **DRNW** | 항산화로 시간 붙잡는 맑은 피부 고양이 | 보습·장벽 → 선크림 → 안티에이징 | 보습형 광범위 |
| **DRNT** | 수분 충전만으로 완성되는 저관리 꿀피부 고양이 | 보습·장벽 → 선크림 | 보습형 광범위 |

> 각 유형이 갖는 필드: 페르소나 · 태그라인 · 요약 설명 · 케어순서 · 자차 · 권장성분 · 기피성분

## 7. 사용법

```dart
import 'package:cosmos_app/features/bsti/bsti.dart';

// 1) 답변: {문항id: 선택한 보기의 score}
final answers = {1: 4, 2: 3, /* … */ 20: 2};

// 2) 진단
final diag = BstiEngine.diagnose(answers);

diag.code;                    // 'OSPW'
diag.type.personaName;        // '진정 먼저 챙기는 풀코스 케어 고양이'
diag.type.careOrder;          // ['진정·장벽', '선크림', '미백', '안티에이징']
diag.recommendedIngredients;  // 권장 성분 (sort 순)
diag.avoidIngredients;        // 기피 성분
diag.sunscreen;               // 배정된 자차
diag.axisScores;              // 축별 점수·극·비율(게이지용)

// 코드만 필요하면
BstiEngine.computeCode(answers);  // 'OSPW'
```

**성분 → 유형 역매칭** (제품·성분 상세의 "권장 피부타입"에 사용)

```dart
// 성분 id들이 어느 유형의 권장 성분과 가장 많이 겹치는지 실제 계산
final type = BstiEngine.matchTypeByIngredients(['cera', 'ha', 'panth']);
type?.code;         // 'DSNW'
type?.personaName;  // '수분으로 주름 늦추는 레이어링 고양이'
```

### 주요 API

| 함수 | 입력 | 출력 |
|:--|:--|:--|
| `BstiEngine.computeCode()` | `Map<int,int>` 답변 | `String` 4글자 코드 |
| `BstiEngine.diagnose()` | `Map<int,int>` 답변 | `BstiDiagnosis` (코드+유형+축점수) |
| `BstiEngine.matchTypeByIngredients()` | 성분 id 목록 | `BstiSkinType?` 매칭 유형 |
