# 푸터(하단바) 아이콘

여기에 아래 이름으로 PNG 를 넣으면 하단바 아이콘이 교체된다.
**없으면 기본 아이콘이 유지**되므로 하나씩 넣어도 된다.

| 파일명 | 탭 | 내용 |
|---|---|---|
| `shelf.png` | 화장대 | 화장품 단품 (립스틱/크림 등 하나) |
| `home.png` | 홈 | 집 |
| `my.png` | 마이 | 사람 |

- **정사각형, 96×96px 이상, 투명 배경** 권장
- 선택 안 된 탭은 같은 이미지를 반투명으로 보여준다.
  선택/비선택을 다른 이미지로 하고 싶으면 `shelf_on.png` 처럼
  `_on` 붙인 파일을 추가로 넣고 말해주면 연결한다.
- 넣은 뒤 앱 재시작 필요 (새 에셋은 핫리로드로 안 잡힘)
